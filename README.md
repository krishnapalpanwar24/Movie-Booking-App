# 🎬 Movie Ticket Booking App

A simple **React.js** app to book movie tickets with seat and showtime selection. Built with **Firebase Firestore** for booking storage.

## 🚀 Live Demo
Check out the app live on Netlify: [Ticket Booking App](https://booking-cinema-app.netlify.app/)

## 🛠 Features
- Browse movies with images and details.
- Select showtime from available slots.
- Choose seats dynamically (already booked seats are disabled).
- Book tickets and store bookings in Firebase.
- See alert confirmation with movie, seat, and time.

## 💻 Tech Stack
- **Frontend:** React.js, React Router
- **Backend / Database:** Firebase Firestore
- **State Management:** React Context API
